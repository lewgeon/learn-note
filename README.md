# MusicNoteTrainer (音名唱名数字转换训练器)

🎵 一个基于 Python 和 Tkinter 开发的轻量级桌面应用程序，旨在帮助音乐初学者和爱好者快速练习并掌握**音名**，**唱名**与**简谱数字**之间的对应关系。

## ✨ 核心特性

* **双轨主模式**：
  * **单符号模式**：随机生成单个符号，进行快速反应训练（自带防连续重复机制）。
  * **符号序列模式**：生成指定长度的符号序列，锻炼综合记忆与转换能力。
* **高自由度定制**：
  * 可自由选择电脑出题的类型（数字、唱名或音名）。
  * 支持自定义答题时间限制（1-60秒）与序列长度（2-10个）。
* **极致顺畅的交互体验**：
  * **非阻塞即时反馈**：答题后无需等待提示框消失，按下回车瞬间清空输入并进入下一题，极致流畅。
  * **智能正则校验**：序列模式支持连续输入（例如看到 `C D E` 可直接输入 `123` 或 `doremi`，无需空格或逗号分割），且全面兼容大小写。
* **实时统计与控制**：
  * 自动统计“正确”与“错误”次数。修改任意配置后，进度与统计数据会自动清零，防止数据污染。
  * 设有趣味的“超时判定”机制，以及随时可用的 **暂停/继续** 功能。
* **模块化架构**：代码结构清晰解耦，分为配置、核心逻辑、界面和入口四个模块，极易进行二次开发与功能扩展（例如引入升降号或和弦）。

## 📁 项目结构

项目采用经典的模块化设计：


## 🚀 安装与运行

本项目使用 Python 原生的 GUI 库 `tkinter` 编写，无需安装任何第三方依赖。

### 前置要求
* **Python 3.x** (建议 Python 3.6 及以上版本)

### 运行步骤
1. 克隆或下载本项目到本地。
2. 打开终端（Terminal / CMD / PowerShell）。
3. 导航到项目所在的文件夹：
   ```bash
   cd 你的路径/MusicNoteTrainer
   ```
4. 运行主程序：
   ```bash
   python main.py
   ```

## 🎮 玩法说明

1. **设置游戏规则**：在界面顶部的“游戏设置”区，选择你想要的模式（单符号/序列）、出题类型，并调整时间限制和序列长度。
2. **开始训练**：点击「开始训练」按钮，计时器启动，屏幕中央会出现题目。
3. **输入答案**：
   * 在输入框中输入另外**两种类型之一**。
   * *示例*：如果出题类型是“音名”（如 `C`），你可以输入数字 `1` 或唱名 `do`。
   * *序列示例*：如果是序列模式（如 `C E G`），你可以直接连续输入 `135` 或 `domisol`。
4. **提交答案**：按下键盘上的 `Enter (回车)` 键提交。输入框旁边会瞬间闪烁提示正确与否，同时题目自动刷新。
5. **暂停与更改**：训练途中点击「暂停」可冻结时间。若中途修改了任何设置，当前游戏会自动中断，需重新点击「开始训练」。
